Fetches Users and Posts from JSONPlaceholder.

Caches fetched data in memory (no database required).

Provides API endpoints for:

Importing data

Listing data

Searching & filtering

Viewing a single item by ID

Proper error handling (network failure, malformed responses, etc.).

Clean folder structure using routes/, services/, and utils/.

---------------setup---------------------------
git clone <your-repo-url>
cd <project-folder>
npm install
npm start server
server runs on http://localhost:3000
